Then you can import and use it:

```ts
import { createComponent, compileReactiveHTMLAsComponentTemplate, bootstrap } from '@lirx/dom';


export interface IAppMainComponentConfig {
  element: HTMLElement;
}

const AppMainComponent = createComponent<IAppMainComponentConfig>({
  name: 'app-main',
  template: compileReactiveHTMLAsComponentTemplate({
    html: `
      Hello world !
    `,
  }),
});

bootstrap(AppMainComponent);
```

This library supports:

- **common-js** (require): transpiled as es5, with .cjs extension, useful for old nodejs versions
- **module** (esm import): transpiled as esnext, with .mjs extension (requires node resolution for external packages)

The library is ready to go and does not require transpiling nor bundling. **However** it has been optimized for three-shacking,
so you should use a bundler like rollup or webpack.


If you prefer to use a CDN, you can use for example [skypack](https://www.skypack.dev/):
[https://cdn.skypack.dev/@lirx/dom](https://cdn.skypack.dev/@lirx/dom)
